二、单层正向算子要点

[B,S,3072]
   | RMSNorm
   |- q_proj -> [B,S,48,128] - q_norm - partial RoPE(64)
   |- k_proj -> [B,S, 8,128] - k_norm - partial RoPE(64) -> KV Cache
   |- v_proj -> [B,S, 8,128] -> KV Cache
   `- GQA Attn -> o_proj -> residual Add
   | RMSNorm
   |- gate (sigmoid+bias) -> TopK=8
   `- experts[w1,w3,SiLU*,w2] x selected -> Combine -> residual Add

每专家 SwiGLU：w1/w3: 3072→1536，w2: 1536→3072。

× 62 layers → BF16: 253952 B/tok (248 KiB) · INT8/FP8: 126976 B/tok (124 KiB)
